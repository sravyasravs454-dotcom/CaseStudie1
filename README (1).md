# AVL Patient Index System

This project implements an AVL Tree for managing patient records in a hospital management system. Patient IDs are stored in a self-balancing AVL Tree to ensure efficient searching, insertion, and deletion operations.

## Features
- AVL Tree insertion
- AVL Tree deletion
- Automatic balancing using LL, RR, LR, and RL rotations
- Inorder traversal of Patient IDs
- AVL Tree height calculation
- Time complexity analysis

## Output
- Displays AVL insertion sequence
- Shows rotations performed during balancing
- Prints final AVL Tree structure
- Performs deletion of selected Patient IDs
- Displays updated AVL Tree after deletions

## Time Complexity
- Search: O(log n)
- Insert: O(log n)
- Delete: O(log n)
- Rotation: O(1)

## Application
This project simulates a MediFlow Hospital Patient Index where patient records can be accessed and updated efficiently using AVL Trees.
